### 👋 Halo, saya `@kiozhu`

> **AI &amp; Automation Engineer.** Saya membangun sistem **otonom yang jalan 24/7** — bot terkontrol penuh via Telegram, pipeline **LLM multi-provider**, dan infrastruktur **trading algoritmik** yang stabil di VPS hemat.

- 🤖 Merancang **otomasi end-to-end**: dari sinyal → eksekusi → monitoring → self-healing.
- 🧠 Mengintegrasikan **LLM (Claude, MiniMax, OpenRouter, dll)** sebagai lapisan keputusan dgn *safety gating* ketat.
- ⚙️ Terobsesi pada **keandalan**: graceful shutdown, watchdog independen, atomic state, anti dobel-instance.
- 📈 Sedang mendalami **sistem kuantitatif market-neutral** &amp; validasi *edge* berbasis data (anti-overfitting).
